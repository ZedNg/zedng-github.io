Loopable Animation (2019)
Unmute to hear sound

We were tasked to create a 35 seconds loopable animation using 5 "characters". 4 of these characters were given us with rigs and we were tasked to create a cohesive plot using these character. I created the box character using Autodesk Maya with elements sculpted using 3D-Coat.

The animation was built and rendered in Autodesk Maya and Arnold Renderer.

The sources of the rigged models are as follows:

  • Ultimate Walker and Ultimate Ball rigs by Ugur Ulvi Yetiskin
  • Flour Sack rig by Joe Daniels
  • Hotdog modelled by Caitlin Sue, rigged by Byron Mallett
  • Jeff by Byron Mallett

The project was part of the course work of MDDN241: 3D Modelling and Animation II for the Bachelor of Design Innovation (BDI) at Victoria University of Wellington (VUW).
