The project brief was about using FDM 3D printing method to create a lampshade for a blue LED. The project required us to use wood and transparent PLA filaments and print 3D objects that were single layer thick and without support and raft. That created a great design challenge as I had to make sure that it allows light to pass through, be aesthetically pleasing, and be structurally feasible for printing.

The project was part of the course work of DSDN104: Digital Creation for the Bachelor of Design Innovation (BDI) at Victoria University of Wellington (VUW).
